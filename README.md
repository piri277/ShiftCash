# ShiftCash
ShiftCash esta pensado para ser una app o sitio web de gestor de gastos, de momento usted podrá encontrar un prototipo NO funcional de como debe verse y fluir el programa.

## Home
Verá una interfaz tipica de los sitios web con información ejemplo de ShiftCash, en la barra de navegacion podrá navvegar a traves de la pagina y así mismo podrá entrar a los paneles de "Inicio de sesion" (login) y "registrate"(register)

### Login
Aquí podrá ver un ejemplo de como será la interfaz de inicio de sesion, como datos pide el correo electronico y la contraseña, tambien en la paret inferior te la opcion de crear una cuenta en caso de no tenerla, si decides salr click a esta opcion te enviará al apartado de "registrate". De darle click al boton de "Iniciar Sesion" se le enviará al dashboard, de momento el programa no tiene ningun tipo de verificacion, como mencione anteriormente es solo un prototipo no funcional

### Register
Aquí podrá ver un ejemplo de como será la interfaz de registro, aqui el usuario podrá crearse una cuenta para esto necesitara ingresar sus datos de:
- Nombre de usuario
- Correo elecgtronico
- contraseña
